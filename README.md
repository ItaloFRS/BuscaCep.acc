***BuscaCep.Acc***

Compartilhando um projeto que desenvolvi durante minha participação na Java & AI Academy, promovida pela Accenture. 
A aplicação consiste em um sistema web que realiza a consulta de CEP, consumindo a API ViaCEP. O back-end, desenvolvido em Java, é responsável por tratar os dados recebidos da API e disponibilizá-los para o front-end em React, onde são exibidos para o usuário de forma simples e intuitiva.

**Tecnologias utilizadas:**

• 🌐 Front-end: React

• ⚙️ Back-end: Java

• 🔗 Integração com API: ViaCEP

Foi uma ótima oportunidade para praticar integração com APIs, organização da comunicação entre front-end e back-end e construção de aplicações web completas.
